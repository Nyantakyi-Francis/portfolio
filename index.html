09 Cryptography • Desktop App

SecureGuard File Encryption Tool

A lightweight desktop file-encryption utility built in C# with a clean Object-Oriented architecture. SecureGuard encrypts and decrypts files using AES-256, producing password-protected .locked files with robust file I/O and clear UI feedback.

  • Problem: Users need a simple, trustworthy workflow to protect sensitive files with password-based encryption.
  • Approach: I built a cross-platform Avalonia UI app with a separated crypto core (AES-256 + SHA-256 key derivation) and chunked streaming I/O for large files.
  • Impact: The project demonstrates end-to-end desktop engineering: event-driven UI, async file processing, and clean abstractions that keep cryptography isolated from the interface.

Key features: drag-and-drop selection, progress tracking, password strength indicator, output folder control, overwrite protection, and timestamp preservation.
